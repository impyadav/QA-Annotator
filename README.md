## About The Project

Making the mammoth task of data annotation for an NLP-based Question-Answering system hassle-free. Just jump right in and start annotating!


<!-- GETTING STARTED -->
## Getting Started
 
### Installation

Install with pip (python 3 required)
   ```sh
   $ pip install QA-Annotator 
   ```
### Start the server
   ```sh
   $ annotator 
   ```
